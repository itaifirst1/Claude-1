# Blackjack Game (Python)

A fully featured, command-line Blackjack game written in Python, designed to closely simulate real casino gameplay while providing detailed player statistics and feedback.

## Overview

This project implements a classic game of Blackjack with standard rules, intelligent dealer behavior, and advanced session tracking. The game focuses on both gameplay accuracy and user experience, making it an excellent example of Python logic, control flow, and state management.

Players begin with a fixed balance and can play multiple rounds while tracking performance metrics such as win rate, profit, and return on investment (ROI).

## Features

- **Classic Blackjack Rules**
  - Hit, Stand, and Double Down options
  - Automatic handling of Blackjack and bust conditions

- **Dealer AI**
  - Dealer follows standard casino rules (hits until 17 or higher)
  - Automated decision-making for fair gameplay

- **Session Statistics Tracking**
  - Win/loss record
  - Win percentage
  - Profit and loss tracking
  - Return on Investment (ROI)

- **Visual & Gameplay Feedback**
  - Animated card draws
  - Clear round-by-round results
  - Informative prompts and messages

- **Bankroll Management**
  - $1000 starting balance
  - Bet handling and balance updates after each round

## Technologies Used
- Standard Python libraries only (no external dependencies)
